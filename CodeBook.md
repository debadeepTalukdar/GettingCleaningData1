# CodeBook

## Dataset
Human Activity Recognition Using Smartphones Dataset

## Variables

- subject: ID of participant
- activity_name: Type of activity

## Measurements
All variables represent mean and standard deviation of:
- Accelerometer data
- Gyroscope data

## Transformations
- Merged training and test datasets
- Extracted mean and std columns
- Replaced activity IDs with names
- Renamed variables for clarity
- Created averaged tidy dataset grouped by subject and activity
